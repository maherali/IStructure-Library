add:withOptions:

Add a set of models to the collection.

- (ISCollection *)add:(id)models withOptions:(NSDictionary *)options

Parameters

models

The models to add to the collection.

options

Options used to add models to the collection.

Return Value

The collection instance.

Discussion

Add a set of models to the collection.

If options does not have a SILENT_KEY key with value YES, an add event is triggered as follows:

$trigger(@"add", $dict(SENDER_KEY, model, OPTIONS_KEY, options, COLLECTION_KEY, self), model);

Passing an index in the AT_KEY key will result in adding the model at the index position. Otherwise, the model is added at the end.

initWithModels:andOptions:

Initialize the collection with a set of models using options.

- (id)initWithModels:(id)models andOptions:(NSDictionary *)options

Parameters

models

A model or an array of models to initialize the collection with.

options

The options used in the initialization.

Discussion

Initialize the collection with a set of models using options.

The models can be either an array of models, an array of dictionaries (attributes for each model), a model, or a dictionary representing the attributes of a model.

After models are added to the models internal array, a reset eent is triggered as follows:

$trigger(@"reset", $dict(SENDER_KEY, self, OPTIONS_KEY, options, COLLECTION_KEY, self) );

At the end, the message initializeWithModels:andOptions: is sent to the instance passing in the models and options as parameters. The base class' implementation is a noop.
