initWithOptions:andStyle:

Initialize the instance of the tableview.

- (id)initWithOptions:(NSDictionary *)options andStyle:(UITableViewStyle)style

Parameters

options

The options used in the initialization of the tableview.

style

The style of the tableview.

Discussion

Initialize the instance of the tableview.

Pass in value for the MODEL_KEY key in options to specify the model attached to the view. Pass in value for the COLLECTION_KEY key in options to specify the collection attached to the view. Usually, you pass in either a model or a collection. A collection have precedence in populating and controlling the appearance of the tableview.
